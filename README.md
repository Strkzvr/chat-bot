# Telegram Bitrix24 Bot

Telegram бот для поиска клиентов в Bitrix24 по номеру телефона или фамилии.

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Настройте переменные окружения в файле `.env`
4. Запустите бот: `npm start`

## Настройка

### Создание Telegram бота
1. Найдите @BotFather в Telegram
2. Создайте нового бота командой `/newbot`
3. Скопируйте токен в `.env` файл

### Настройка Bitrix24
1. Перейдите в Bitrix24 → Приложения → Разработчикам
2. Создайте локальное приложение
3. Получите Client ID и Client Secret
4. Настройте OAuth авторизацию для получения токенов

## Команды бота

- `/start` - Начать работу
- `/auth` - Авторизация
- `/phone <номер>` - Поиск по телефону
- `/lastname <фамилия>` - Поиск по фамилии
- `/help` - Справка
